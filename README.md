# Kukreja's Consultancy Website

A professional business consultancy website built with React and Tailwind CSS.

## Features

- Modern, responsive design
- Professional business sections (Home, Services, About, Contact)
- Contact form
- Tailwind CSS styling
- Mobile-friendly navigation

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm start
```

The website will open at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```

## Technologies Used

- React 18
- Tailwind CSS 3
- PostCSS
- Autoprefixer

## Customization

You can customize the colors in `tailwind.config.js`:
- Primary color: Navigation and headings
- Secondary color: Buttons and accents
- Accent color: Call-to-action buttons
